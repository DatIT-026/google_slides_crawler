# Google Slides Crawler (GSC)

A Chrome Extension that automates the extraction of high-quality Google Slides and notes into a clean, structured PDF.

![gsc_image_1](/img/gsc_image_1.png)

## ✨ Features

- **One-Click Export**: Converts slides to PDF directly from the browser.

- **High Quality**: Captures high-resolution slide images.

- **Note Support**: Option to include author's notes below each slide.

- **Smart Formatting**: Automatic A4 layout with justified text for notes.

## 🛠️ Installation

Since this extension is in development, install it manually:

- Clone this repo or download the extension [here](https://github.com/DatIT-026/google_slides_crawler/releases/).

- Open **Chrome** and navigate to `chrome://extensions/` (or `edge://extensions/` for **Edge**)

- Enable **Developer mode** (toggle in the top right).

- Click **Load unpacked** and select the project folder.

## 🚀 How to use?
### Step 1: Prepare the URL

Open your Google Slides presentation and change the URL ending from `/edit...` to `/htmlpresent`.

For example:

- **Original**: `.../presentation/d/1abc.../edit`.

- **Modified**: `.../presentation/d/1abc.../htmlpresent`.

### Step 2: Export
Click the Google Slides Crawler icon in your extension toolbar. Click **Export PDF**. 

Wait for the progress bar to finish. The PDF will download automatically.
